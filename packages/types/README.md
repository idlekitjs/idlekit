# @idlekitjs/types

Shared TypeScript contracts for IdleKit packages. This package contains engine
interfaces and save contracts with no runtime implementation.

## Install

```sh
pnpm add @idlekitjs/types
```

## Import

```ts
import type { Extension, SaveAdapter, System } from "@idlekitjs/types";
```

## Status

Public package under the `@idlekitjs` scope. Publish-ready, but not published to npm yet.
