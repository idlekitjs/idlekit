# @idlekitjs/utils

Framework-agnostic helpers shared across IdleKit packages and games.

## Install

```sh
pnpm add @idlekitjs/utils
```

## Import

```ts
import { clamp, FrameRateMeter, positiveOr } from "@idlekitjs/utils";
```

## Status

Public package under the `@idlekitjs` scope. Publish-ready, but not published to npm yet.
